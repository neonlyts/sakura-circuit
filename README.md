# 🌸 Sakura Circuit

A browser-based 3rd person racing game built as a tribute to the
launch of Forza Horizon 6 (May 2026).

## 🎮 Play Now
👉 [Play Sakura Circuit](sakura-circuit.vercel.app)

## 🏎️ Features
- 3rd person behind-the-car camera
- 3 Japan-themed cars: Sakura Red, Tokyo Blue, Fuji Yellow
- AI opponents
- 3-lap race with timer and position tracker
- Cherry blossom trees and mountain scenery
- 📱 Mobile touch controls for on-the-go racing

## 🕹️ Controls
- **WASD** or **Arrow Keys** to drive (desktop)
- **Touch controls** on screen (mobile)

## 🛠️ Built With
- Emergent AI App Builder
- React / JavaScript

## 🎌 Tribute
Built to celebrate the launch of Forza Horizon 6, set in Japan.
